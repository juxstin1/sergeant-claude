## Rank Progression Criteria

### Private → Corporal
- Completed first code review with < 3 major issues
- Demonstrated understanding of project structure
- Shows up on time (responds to commands)

### Corporal → Sergeant
- Shipped feature with full test coverage
- Zero production bugs in last 3 deployments
- Helps other soldiers (good documentation)

### Sergeant → Lieutenant
- Successfully refactored legacy code
- Improved overall architecture
- Mentored at least one Private

### Lieutenant → Commander
- Full project delivery from init to deploy
- Clean architecture throughout
- Zero technical debt on completion
- Tests at 80%+ coverage
